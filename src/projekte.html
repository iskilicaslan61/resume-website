Projekte

Projekt 1: Microservices-Architektur mit Kubernetes

Beschreibung: Entwicklung einer automatisierten Anwendung auf Basis der Microservices-Architektur mit Java, Spring Boot und Spring Cloud, integriert mit MySQL.

Technologien: Java, Spring Boot, Docker, Kubernetes, Rancher, Jenkins, Terraform, AWS EC2, Prometheus, Grafana

Ergebnisse: Vollständig containerisierte und orchestrierte Microservices-Anwendung mit automatisierten CI/CD-Pipelines und Monitoring.

Projekt 2: Cloud-Native Blog-Anwendung auf AWS

Beschreibung: Entwicklung und Bereitstellung einer Blog-Webanwendung auf AWS Cloud Infrastructure, basierend auf dem Django Framework mit Load Balancing und Auto Scaling.

Technologien: Django, AWS EC2, RDS, S3, Lambda, DynamoDB, CloudFront, Route53, ALB, Auto Scaling Group

Ergebnisse: Skalierbare, hochverfügbare Cloud-Anwendung mit automatischer Lastverteilung und serverlosen Funktionen.

Projekt 3: Infrastructure as Code mit Terraform

Beschreibung: Automatisierte AWS-Infrastruktur für statische Webseiten mit S3, CloudFront, Route53 und ACM-Zertifikaten, vollständig mit Terraform verwaltet.

Technologien: Terraform, AWS S3, CloudFront, Route53, ACM, GitHub Actions, CI/CD

Ergebnisse: Vollständig automatisierte, skalierbare und sichere Cloud-Infrastruktur mit automatischen Deployments.

Projekt 4: Kubernetes Phonebook Microservice

Beschreibung: Entwicklung einer Phonebook-Microservice-Webanwendung mit MySQL-Datenbank, um die Microservice-Architektur praktisch umzusetzen.

Technologien: Docker, Kubernetes, MySQL, Frontend/Backend Services, Container Orchestration

Ergebnisse: Containerisierte Microservices-Anwendung mit Kubernetes-Deployments und skalierbarer Architektur.
